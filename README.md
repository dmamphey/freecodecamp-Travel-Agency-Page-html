# Travel Agency Page

A simple HTML project built as part of my freeCodeCamp coding practice.

## About the Project

This project is a basic travel agency webpage called **Discover Italy**. It promotes travel packages and top itineraries for visitors interested in exploring Italy.

The page includes:

- A main heading and introductory paragraph
- A section for travel packages
- Links for group travel and private tours
- A top itineraries section with images and captions

## Features

- Clear page structure using HTML
- Travel package links
- Clickable images
- Figure and figcaption elements
- Alt text for accessibility
- Simple and readable content layout

## Built With

- HTML5

## What I Learned

While building this project, I practised:

- Creating a basic HTML document
- Using headings and paragraphs
- Adding links with the anchor tag
- Building unordered lists
- Embedding images in a webpage
- Using `figure` and `figcaption`
- Writing alt text for accessibility
- Structuring content clearly with HTML
